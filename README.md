# Tubes Web

Project ini merupakan tugas besar web yang saat ini berada pada tahap progres 1.

## Deskripsi Progres 1
Pada tahap ini, fitur-fitur yang telah diimplementasikan adalah:

1. **Login Page dan Register**: Halaman untuk autentikasi pengguna.
2. **CRUD**: Fitur Create, Read, Update, Delete sudah tersedia.
3. **Database**: Menggunakan MySQL dengan XAMPP sebagai environment.

## Catatan
* Belum mengimplementasikan **Session** dan **Cookie**.
